# VLSI Internship - Task 2
## Synchronous RAM Design

### Description
This task involves the design of a parameterizable Single-Port Synchronous RAM in Verilog. 

### Specifications
- **Data Width:** 8-bit
- **Address Width:** 4-bit (16 locations)
- **Clocking:** Synchronous Read/Write on the rising edge.
